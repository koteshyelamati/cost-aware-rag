MongoDB Atlas Vector Search uses HNSW indexing for approximate nearest
neighbor search. It supports cosine, euclidean, and dot product similarity.
The $vectorSearch aggregation stage is used in queries.
Redis is an in-memory data store used for caching and queues.
Semantic caching stores query embeddings and reuses answers for similar
queries. Cosine similarity measures angle between vectors.
Gemini Flash is a lightweight fast model good for simple queries.
Gemini Pro is more powerful and expensive, used for complex queries.
